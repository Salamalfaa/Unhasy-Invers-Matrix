# Invers Matriks

A simple interactive matrix inverse calculator built with plain HTML, CSS, and JavaScript.

## Fitur

- Hitung invers matriks 2x2, 3x3, dan 4x4
- Tampilan tema gelap/terang
- Waktu WIB real-time di header
- Tombol `Contoh Isian` menghasilkan angka acak baru setiap klik
- Popup informasi proyek dengan tombol minimize dan close
- Minimize menyembunyikan popup dan meninggalkan ikon kecil
- Close menutup popup dan menghilangkan ikon popup

## Cara Menggunakan

1. Buka file `matrix_inverse_calculator.html` di browser.
2. Pilih ukuran matriks (2x2, 3x3, 4x4).
3. Klik `Contoh Isian` untuk mengisi nilai acak.
4. Klik `Hitung Invers` untuk melihat langkah dan hasil invers.
5. Klik `Reset` untuk mengosongkan matriks.
6. Gunakan tombol tema untuk beralih antara mode terang dan gelap.

## Struktur Folder

- `matrix_inverse_calculator.html` — halaman utama aplikasi
- `assets/css/style.css` — styling aplikasi
- `assets/js/app.js` — logika interaktif aplikasi

## Catatan

- Pastikan semua file berada dalam folder yang sama seperti struktur di atas.
- Aplikasi tidak memerlukan server khusus, bisa dibuka langsung di browser.
